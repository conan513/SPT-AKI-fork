# SPTarkov-Server

Modding framework for Escape From Tarkov

## Requirements

- VSCodium
- NodeJS 12.18.2
- Escape From Tarkov 0.12.7

## Setup

VSCodium -> menu bar -> tasks -> run task... -> NPM -> install

## Build

VSCodium -> menu bar -> tasks -> run build task
