# Server

Modding framework for Escape From Tarkov

## Requirements

- VSCodium
- NodeJS 12.18.2
- Escape From Tarkov 0.12.9

## Setup

VSCodium > tasks > run task... > NPM > install

## Build

VSCodium > tasks > run build task
